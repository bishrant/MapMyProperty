The Draw toolbox contains tools to draw areas (polygons and circles), lines (linear features), and points of interest for a boundary.  In addition, text can be added to the map.

To define areas, circles, linear features, and points, or to add text to the map, select the button corresponding to the drawing tool of interest to activate it and all the drawing options to style that element will be available.

For example, to draw a boundary, select the Polygon button and select the options for drawing mode, line style, and fill style.  Once these are set, move the cursor to the map and begin adding points at each location where the boundary changes direction.  Move the cursor in the direction of the next point and click to add a new point.  Continue this until you draw the entire boundary.  Double-clicking to add the last point will complete the drawing.

To add circles, lines, points of interest, or text, follow the same steps as for area polygons.  To draw a circle, select the Circle radio_button_unchecked tool and specify the drawing mode.  Two options are offered, Click and Freehand.  By selecting Freehand you can draw a circle by hand.  To use this option, move the cursor to the map, and press down the left button of the mouse at the location that will be the center of the circle, and start moving the cursor away from the center to define its size.  When you are satisfied with the size of the circle, release the mouse button.  To create a circle with a specific radius distance, select the Click drawing mode option, enter the desired radius, move the cursor to the map and click on the map to specify the center of the circle.

To draw a line, select the Line show_chart tool and specify the drawing mode and style options.  Linear features can be drawn either as polylines (connected segments of straight lines) or as freehand lines, such as the way you draw using a pencil.

To add points of interest or text, select the respective tool and specify the appearance options as was done for areas and lines.  Click the Point or Text tool, whichever is applicable, and click anywhere on the map to create the point or text.

To deactivate all the drawing tools click on the Clear button whenever a tool is active.

If you are not satisfied with the drawing and want to edit it, click on the drawn feature to select it and enable editing mode.  In the case of editing an area, notice the blue circular points on each corner or vertex.  Click on these and drag them to a new, desired location to move these points.  To delete an existing vertex, right-click on it.  New vertices can be added by clicking on and dragging the "ghost" vertices (grey circular points) to the desired location.  The moved ghost vertex will now show as a new point (blue).  Ghost vertices are displayed halfway between two existing points.  Similar steps can be followed to edit lines.

Points of interest can be moved by first clicking on it to select it, which will be indicated by changing to cyan blue color.  You can now click and drag it to a new location.

Text can be moved by first clicking on it and then dragging the text to the desired position using the input box handler.  To delete the text click on the delete icon.

If you want to move a label showing the acreage, click on the label to select it, then click and drag it to the new location.  If you do not wish to display the acreage labels, uncheck the Area/Distance Labels checkbox, and vice versa.

Clicking on the Delete All button (at the bottom of the map) will delete all drawings.  Clicking the Undo button (also at the bottom of the map) will delete the last feature drawn.  Clicking on it again will delete the next to last one drawn.  This can be done to sequentially delete features in reverse order that they were added.  The same functionality holds for the Redo button.
