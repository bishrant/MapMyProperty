The Swipe Layers window is used to interactively compare two different imagery layers for the state of Texas.

To use the tool, select the appropriate layer in the dropdown menus and the "Swipe Direction" option on the bottom.  The selected "Swipe Direction" provides a way to interact with the map and there are 2 options:

Left-Right:  It is the default style.  With this option selected, you slide the swipe tool left and right.

Top-Bottom:  With this option selected, you slide the swipe tool up and down.
